Learning Sudoku

Master the art of Sudoku with our comprehensive tutorials and strategies

What is Sudoku?

Sudoku is a logic-based number placement puzzle. The objective is to fill a 9×9 grid with digits so that each column, each row, and each of the nine 3×3 subgrids (also called "boxes" or "regions") contains all of the digits from 1 to 9.

Rules of Sudoku

  1. Each row must contain the numbers 1-9 without repetition.
  2. Each column must contain the numbers 1-9 without repetition.
  3. Each of the nine 3×3 subgrids must contain the numbers 1-9 without repetition.

How to Start

When you start a Sudoku puzzle, some cells will already be filled with numbers. These are called "givens" or "clues". The more givens in a puzzle, the easier it typically is to solve.

Beginner Tip

Start by scanning rows and columns to identify where a specific number can be placed based on its absence elsewhere. Look for rows, columns, or boxes that already have several numbers filled in.

Basic Techniques

1. Scanning

Systematically check each row, column, and box to identify missing numbers and potential placements.

2. Pencil Marking

Write down all possible numbers for each empty cell. As you fill in cells, update and eliminate options from remaining cells.

Intermediate Techniques

3. Candidate Elimination

If a number appears as a candidate in only one cell within a row, column, or box, then that number must go in that cell.

4. Pointing Pairs/Triples

When a number can only appear in two or three cells within a box, and these cells are all in the same row or column, that number can be eliminated from other cells in that row or column.

Advanced Techniques

5. X-Wing

When a candidate appears in exactly two cells in each of two different rows, and these cells also align in the same columns, the candidate can be eliminated from other cells in those columns.

6. Swordfish

An extension of X-Wing involving three rows and three columns.

Sudoku Variants

Beyond classic Sudoku, there are many exciting variants that add new rules and challenges:

Classic Sudoku

The standard 9×9 grid with the three basic rules.

Killer Sudoku

Combines Sudoku with elements of Kakuro. Cells are grouped into "cages" with specified sums, and no digit can be repeated within a cage.

Diagonal Sudoku

In addition to the standard rules, the two main diagonals of the grid must also contain the digits 1-9 without repetition.

Odd/Even Sudoku

Some cells are shaded to indicate that they must contain odd or even digits.

Non-Standard Regions

Instead of the usual 3×3 boxes, the grid is divided into nine irregularly shaped regions.
